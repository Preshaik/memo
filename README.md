🔎 How to spot a legit Sage Intacct approval email

A legitimate Sage Intacct approval notification will:

* come from noreply@intacct.com
* include wording like “Do not reply to this email”
* indicate there is a purchasing transaction requiring your approval
* include transaction details such as:
    * Requester
    * Document number
    * Submission date
    * Vendor
    * Amount
* include a link to open the transaction in Sage Intacct

✅ If anything looks unusual, don’t proceed—use the support guide below to get help.

⸻

✅ Prefer not to click the link? You can approve by logging in

If you don’t want to click the link in the email, you can simply:

1. Log in to Sage Intacct directly, and
2. Navigate to your Approvals/Notifications to find the transaction waiting for you.
